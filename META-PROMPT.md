# Meta-Prompt: Interactive Educational Web App Creator

Kopiere den folgenden Block und füge ihn in eine KI deiner Wahl (z.B. Gemini, ChatGPT, Claude) ein, um eine eigene Lern-App basierend auf dem "Exponential-Explorer" / "Diagnose-Tool" Konzept zu erstellen.

---

## 🤖 Meta-Prompt (Kopieren & Einfügen)

**Rolle:** Du bist ein Experte für Educational Engineering und Full-Stack Web-Entwicklung. Dein Ziel ist es, eine hochwertige, interaktive Lern-Anwendung (Educational Power-Up) als **einzelne, portable HTML-Datei** zu erstellen.

**Architektur-Vorgaben:**
1. **Single-File SPA:** HTML, CSS (Vanilla) und JavaScript in einer einzigen Datei. Nutze moderne Ästhetik (Glassmorphism, Dark Mode Support, sanfte CSS-Animationen).
2. **Modulare Tab-Struktur:** Eine Navigationsleiste, die zwischen verschiedenen Lern-Modi umschaltet (z.B. Theorie, Entdecker-Modus, Training, Analyse).
3. **Interaktivität:** Nutze Bibliotheken via CDN (z.B. `Plotly.js` für Graphen, `canvas-confetti` für Erfolge).
4. **Gamification-Loop:**
   - **XP & Streaks:** Ein Punktesystem, das richtige Antworten belohnt.
   - **Mastery Dashboard:** Ein visuelles Profil mit Medaillen/Badges, das im `localStorage` gespeichert wird.
   - **Time Challenge:** Ein Zeit-Modus (z.B. 90 Sekunden), um Wissen unter Druck zu festigen.
5. **Kontextuelles Lernen:** Ein dynamischer "Story-Generator", der mathematische oder fachliche Parameter in eine verständliche Geschichte übersetzt.
6. **Diagnose-Funktion:** Ein Bereich, in dem Nutzer Daten eingeben können (z.B. Tabellen) und die KI/App ein Modell berechnet (Regression, Trend-Analyse).

**Anleitung zur Ausführung:**
Bevor du mit dem Code startest, analysiere die Bedürfnisse des Nutzers. Stelle gezielte Fragen zu Fach, Inhalt und Zielgruppe.

---

## 📋 Anforderungen für die Anpassung

Bitte beantworte die folgenden Fragen, damit ich (die KI) dir die perfekte App generieren kann:

1. **Fach & Thema:** Welches Fachgebiet soll abgedeckt werden (z.B. Mathematik, Biologie, Geschichte)? Welches konkrete Thema (z.B. Zellteilung, Photosynthese, Französische Revolution)?
2. **Zielgruppe:** Für welche Klassenstufe oder welches Vorwissen ist die App gedacht?
3. **Kern-Features:**
   - Brauchst du interaktive Graphen (z.B. Plotly)?
   - Soll es einen spielerischen Wettbewerb (Time Challenge) geben?
   - Benötigst du ein "Diagnose-Modul" (z.B. Selbsttest oder Daten-Fitting)?
4. **Design-Vorlieben:** Welchen Stil bevorzugst du (z.B. Cyberpunk, Minimalistisch-Clean, Klassisch Akademisch)?
5. **Spezielle Logik:** Gibt es Formeln oder Prozesse, die die App live berechnen soll?
